# doubanSpider
##### 这是一个爬虫的项目，爬取豆瓣的图书，作为我们Python课程的作业

## 安装
#### 缺少什么Python库就用什么安装什么库，pip也行，conda也行
## 使用
#### 直接每一个py文件都可以用来运行
## 代码解释：文中有注释
* |
* spider--spidersGo.py 开始一个爬虫，可以爬取多种类型的书籍
* |--------spiderAio 查询某一种类的图书，同时查询多个页面
* |--------numberOfPeople  查询某一图书的评价人数
* |
* sql------conn 连接数据库
* |--------connj.json 连接数据库的各种参数
